# ua-damage-map-api
This is a NestJS REST API backend for Damage Map, a project created for [
Hackathon4Ukraine](facebook.com/events/1132043360669520).

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```
